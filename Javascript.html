Basic JavaScript Syntax:

  • Variables: Used to store data values. Declared using var, let, or const. Example: let x = 5;
  • Functions: A block of code designed to perform a particular task. Example: function greet() { console.log('Hello!'); }
  • Conditionals: Used to perform different actions based on different conditions. Example: if (x > 10) { console.log('x is greater than 10'); }
  • Loops: Used to repeat a block of code. Example: for (let i = 0; i < 5; i++) { console.log(i); }
  • Arrays: Used to store multiple values in a single variable. Example: let fruits = ['apple', 'banana', 'cherry'];
  • Objects: Used to store collections of data and more complex entities. Example: let person = { name: 'John', age: 30 };
  • Events: Actions that occur in the browser, such as clicks or key presses. Example: element.addEventListener('click', function() { alert('Clicked!'); });
  • DOM Manipulation: Used to interact with and modify HTML elements. Example: document.getElementById('container').innerHTML = 'Hello World';
  • Arrow Functions: A shorter syntax for writing functions. Example: const add = (a, b) => a + b;
  • Promises: Used to handle asynchronous operations. Example: fetch('url').then(response => response.json()).then(data => console.log(data));
