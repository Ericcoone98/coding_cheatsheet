Basic JavaScript Syntax:

  • Variables
    Used to store data values. Declared using var, let, or const. Example: let x = 5;
  • Functions
    A block of code designed to perform a particular task. Example: function greet() { console.log('Hello!'); }
  • Events
    Actions that occur in the browser, like clicks or key presses. Example: document.addEventListener('click', function() { alert('Clicked!'); });
